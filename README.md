# Growth Marketing AI Toolkit

## Overview

This repository documents the setup and configuration of modern AI-assisted development tools, including Cursor IDE, Claude Code, Codex, and GitHub.

The project was completed as part of an application process for a Junior Growth Marketing Specialist role. The objective was not only to install the required tools, but also to demonstrate the ability to learn unfamiliar technologies, document processes, and solve problems independently.
